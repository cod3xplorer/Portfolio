Este projeto em Power BI apresenta uma solução completa de análise comercial, com foco em:

📈 Atingimento de metas

👥 Performance de vendedores

📦 Performance de produtos

🔮 Projeção de resultados (Forecast)

🤖 Recomendação de produtos baseada em comportamento

O dashboard foi construído para apoiar decisões estratégicas, combinando análise descritiva, diagnóstica e preditiva.

🧠 Principais Insights do Projeto

Identificação de meses abaixo da meta

Comparação entre períodos distintos

Avaliação de vendedores (quem performa vs quem precisa melhorar)

Relação entre faturamento e margem

Projeção de atingimento de meta no mês

Sugestão de produtos com base em clientes similares

Principais análises:

KPIs:

Faturamento

Margem

Ticket Médio

Evolução temporal do faturamento

Comparação entre períodos (Ano atual vs anterior)

Identificação automática de meses abaixo da meta

Destaques:

Ranking de vendedores

% de atingimento da meta individual

Comparação mês a mês

Crescimento vs Atingimento

Alta performance

Baixo crescimento

Risco comercial

📦3. Performance dos Produtos
Análises disponíveis:

Top produtos por faturamento

Produtos com maior/menor margem

Relação:

Faturamento vs Margem (%)

Distribuição de preços

Identificação de oportunidades:

Produtos rentáveis vs volume

🔮4. Forecast (Projeção de Meta)
Recursos:

Projeção de faturamento até o fim do período
